# Contributing to RealAgentID

Thanks for your interest in contributing. RealAgentID is an open source cryptographic identity system for AI agents.

## Getting Started

'''bash
- git clone https://github.com/RealAgentID/RealAgentID.git
- cd RealAgentID
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt

'''bash

#### How to Contribute

- Fork the repo
- Create a branch: git checkout -b feature/your-feature
- Make your changes
- Run the test suite
- Submit a pull request

##### Areas I Need Help

- New agent adapters (healthcare, legal, infrastructure)
- SDK development (pip install realagentid)
- Attack simulations and demos
- Documentation improvements

###### Code Style

- Python 3.12+
- Keep functions small and single purpose
- Add docstrings to new functions
- All new features need to be tested
